# Install dependent environment

```shell
npm install
```

# Testing

run Hardhat Network
```shell
npx hardhat node
```


deploy these contracts:

```shell
hardhat run --network localhost scripts/deploy.ts
```

run the test suite:
```shell
npx hardhat test --network localhost
```
